# ESG Readiness & Governance Gap Analyzer
*(working name: ESG Regulatory-Gap Tracker)*

**What it is:** An AI-assisted workflow that compares a company's ESG disclosures against
a regulatory checklist (ADGM/UK-style) and produces a structured gap report — directly
extending the ESG advisory engagement you delivered at Corporate Advisory Group.

**Why it matters:** ESG regulatory requirements are still fragmented and evolving; most
gap analysis is done manually against dense disclosure requirements. This workflow
turns that into a repeatable, auditable process.

## Files

- `prompt-template.md` — the reusable prompt
- `sample-disclosure.md` — a synthetic company ESG disclosure excerpt
- `sample-output.md` — the generated gap report
- `cv-summary.md` — CV line, LinkedIn post, interview talking points
- `real-example-adib.md` — the tracker run against Abu Dhabi Islamic Bank's real, public ESG overview page — notable for correctly identifying that a landing page can't be fully assessed as a complete disclosure, rather than overstating gaps from thin material
